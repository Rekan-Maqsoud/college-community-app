Delete Account Proxy (Appwrite Function)

Purpose:

- Perform secure, server-side account deletion cleanup with API key privileges.
- Remove user-generated data and anonymize the user profile to "Deleted Account".

Required env vars:

- APPWRITE_ENDPOINT
- APPWRITE_PROJECT_ID
- APPWRITE_API_KEY
- APPWRITE_DATABASE_ID
- APPWRITE_USERS_COLLECTION_ID
- APPWRITE_POSTS_COLLECTION_ID
- APPWRITE_REPLIES_COLLECTION_ID
- APPWRITE_CHATS_COLLECTION_ID
- APPWRITE_MESSAGES_COLLECTION_ID
- APPWRITE_USER_CHAT_SETTINGS_COLLECTION_ID
- APPWRITE_NOTIFICATIONS_COLLECTION_ID
- APPWRITE_PUSH_TOKENS_COLLECTION_ID
- APPWRITE_LECTURE_MEMBERSHIPS_COLLECTION_ID
- APPWRITE_LECTURE_ASSETS_COLLECTION_ID
- APPWRITE_LECTURE_COMMENTS_COLLECTION_ID
- APPWRITE_REP_VOTES_COLLECTION_ID
- APPWRITE_SUGGESTIONS_COLLECTION_ID

Optional env vars:

- APPWRITE_REP_ELECTIONS_COLLECTION_ID

Client app env:

- EXPO_PUBLIC_DELETE_ACCOUNT_ENDPOINT

Auth:

- Requires Authorization: Bearer <JWT> from the signed-in user.
- Supported action: delete_account
